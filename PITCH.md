# 3-Minute Pitch

BountyAgent turns an AI agent into a paid digital worker.

The problem is that most AI agents only chat. They do not have identity, payment rails, or proof that they completed work. BountyAgent demonstrates a simple agent economy loop: a user requests a microtask, the agent quotes a low price, receives payment through x402, completes the task, verifies the result, and returns a receipt tied to its ERC-8004 identity.

For the demo, BountyAgent generates a verified lead pack. The user asks for relevant opportunities. BountyAgent returns three structured leads, each with a role, company, location, fit reason, and next action. VerifyAgent checks that the deliverable has the required fields. The final output includes a receipt with agent identity, payment status, transaction reference, verification status, and timestamp.

This is not charging for generic AI text. It is charging for completed, verified micro-work.

Long term, BountyAgent can become a reusable template for paid autonomous agent services: grant searches, supplier research, compliance checks, recruiting leads, document review, and agent-to-agent subcontracting.
